// == ローカルストレージ
// ローカルストレージの機能では、メールメッセージをクラウドストレージからご利用中の端末のローカルストレージへ移動することができます。

// ご注意：この機能は {product-name} のデスクトップアプリでのむ提供しています。

// === 古い {product-name} アプリから新しいバージョンへローカルストレージのフォルダーを移管する
// ご注意：古いバージョンでローカルのフォルダーを利用した場合、以下の手順で新バージョンへ移管してください。

// ==== 空のフォルダーを作成する
// . {product-name} のデスクトップアプリをインストールし、起動します。
// . ご利用アカウントへログインします。
// . 左側にて、 *Local Storage* の上にマウスを置き、*+* をクリックし、新しいフォルダーを作成します。
// +
// ご注意：古いフォルダーはローカルストレージに表示しない。以下の手順で古いメールをローカルストレージにある新しいフォルダーへインポートします。
// +
// . ファイルー名を入力します。以下の事例として、このフォルダーを *NewLocalFolder* としました。

// ==== ファイルをコピーする
// . 古いフォルダーをアクセスします：
// + 
// Windows の場合:: `%APPDATA%\@{product-name}\<メールアドレスのフォルダー名>\Mail\<古いアプリのローカルフォルダー>`
// Mac の場合:: `~/Library/Application Support/@{product-name}/<メールアドレスのフォルダー名>/Mail/<古いアプリのローカルフォルダー>`
// +
// . フォルダー内の `EML` ファイルをすべてコピーする
// . `@{product-name}/storage/<メールアドレスのフォルダー名>/Mail/NewLocalFolder` をアクセスします。
// . コピーしたファイルを貼り付けます。

// ===== 実例
// 以下の条件とします：

// * メールアドレスは ada@example.com
// * 古い {product-name} のデスクトップアプリでは、ローカルフォルダー名は`OldLocalFolder`
// * 新しい {product-name} のデスクトップアプリでは、ローカルフォルダー名は`NewLocalFolder`

// 上記の場合、`EML` ファイルは以下のようにコピーします：

// .File path
// |===
// |OS |元の場所 |新しい場所 

// |Windows
// |`%APPDATA%\@{product-name}\\ada@example.com\Mail\OldLocalFolder`
// |`%APPDATA%\@{product-name}\storage\\ada@example.com\Mail\NewLocalFolder`

// |Mac
// |`~/Library/Application Support/@{product-name}/ada@example.com/Mail/OldLocalFolder`
// |`~/Library/Application Support/@{product-name}/storage/ada@example.com/Mail/NewLocalFolder`
// |===

// ==== メールをインポートする

// . {product-name} のデスクトップアプリをアクセスします。
// . *Local Folder* の直下にあるフォルダーをクリックします -- 実例の場合は`NewLocalFolder`です
// . フォルダー (`NewLocalFolder`) の上にマウスを置きますと、リフレッシュのアイコンが表示します。
// . リフレッシュのアイコンをクリックし、古いアプリからコピーしたメールが新しいフォルダー(`NewLocalFolder`)で閲覧することが可能となります。

// == ローカルのゴミ箱フォルダーについて
// ローカルのゴミ箱フォルダーは *Local Folders* で削除したメールやフォルダーを保管します。ゴミ箱では、メールを閲覧済みにマークするか、フォルダー内のメッセージをすべて削除することが可能です。

// === 既読としてマークする
// このオプションでは、ローカルのゴミ箱フォルダーにあるすべてのメールを既読としてマークします。


// . *Local Folders*の直下にある *Trash* を右クリックします。
// . メニューから *Mark emails as read* をクリックします。
// . すべてのメッセージが既読としてマークされます。

// === ゴミ箱フォルダーを空にする
// このオプションでは、ローカルのゴミ箱フォルダーにあるすべてのメールを削除します。

// . *Local Folders*の直下にある *Trash* を右クリックします。
// . メニューから *Empty trash* をクリックします。
// . すべてのメッセージが削除されます。

=== ローカルフォルダーを管理する

// ご注意：この機能は Windows 用の {product-name} デスクトップアプリでのみ提供しています。

ローカルフォルダーを活用することで、古いメールやサイズが大きいメールを削除せずに、クラウド上に割り当てた容量の上限を保護することが可能です。メールをクラウドからローカルのストレージへ移動し、メールをご利用中のコンピュータ上で保管することになります。

=== 複数ローカルフォルダーを作成する
. 左側にて、*Local Storage* の上にマウスを置きます。
. image:graphics/plus.svg[plus icon, width=20] のアイコンをクリックし、新しいフォルダーを作成します。
. フォルダー名を入力し、キーボードの Enter キーを押します。
. 新しく作成したフォルダーをローカルストレージ上で確認できます。

=== ローカルフォルダーのフォルダー名を変更する
. *Local Storage*の直下にあるフォルダーを右クリックします。
. メニューから *Rename* をクリックします。
. フォルダーの新しい名を入力します。
. キーボードの Enter キーを押しますとフォルダー名が新しい名に変更されます。

=== ローカルフォルダーをローカルのゴミ箱へ削除する
. *Local Storage*の直下にあるフォルダーを右クリックします。
. メニューから *Delete* をクリックします。
. 確認画面にて、*OK* をクリックします。
. 削除したフォルダーがローカルフォルダーで *Trash* へ移動されます。
endif::Desktop_app[]
