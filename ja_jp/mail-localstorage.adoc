// == ローカルストレージ
// ローカルストレージの機能では、メールメッセージをクラウドストレージからご利用中の端末のローカルストレージへ移動することができます。

// IMPORTANT: この機能は {product-short} のデスクトップアプリでのみ提供しています。

// === 古い {product-short} アプリから新しいバージョンへローカルストレージのフォルダを移管する
// IMPORTANT: 古いバージョンでローカルのフォルダを利用した場合、以下の手順で新バージョンへ移管してください。

// ==== 空のフォルダを作成する
// . {product-short} のデスクトップアプリをインストールし、起動します。
// . ご利用アカウントへログインします。
// . 左側にて、 *Local Storage* の上にマウスを置き、*+* をクリックし、新しいフォルダを作成します。
// +
// NOTE: 古いフォルダはローカルストレージに表示しない。以下の手順で古いメールをローカルストレージにある新しいフォルダへインポートします。
// +
// . ファイルー名を入力します。以下の事例として、このフォルダを *NewLocalFolder* としました。

// ==== ファイルをコピーする
// . 古いフォルダをアクセスします：
// + 
// Windows の場合:: `%APPDATA%\@{product-short}\<メールアドレスのフォルダ名>\Mail\<古いアプリのローカルフォルダ>`
// Mac の場合:: `~/Library/Application Support/@{product-short}/<メールアドレスのフォルダ名>/Mail/<古いアプリのローカルフォルダ>`
// +
// . フォルダ内の `EML` ファイルをすべてコピーする
// . `@{product-short}/storage/<メールアドレスのフォルダ名>/Mail/NewLocalFolder` をアクセスします。
// . コピーしたファイルを貼り付けます。

// ===== 実例
// 以下の条件とします：

// * メールアドレスは ada@example.com
// * 古い {product-short} のデスクトップアプリでは、ローカルフォルダ名は`OldLocalFolder`
// * 新しい {product-short} のデスクトップアプリでは、ローカルフォルダ名は`NewLocalFolder`

// 上記の場合、`EML` ファイルは以下のようにコピーします：

// .File path
// |===
// |OS |元の場所 |新しい場所 

// |Windows
// |`%APPDATA%\@{product-short}\\ada@example.com\Mail\OldLocalFolder`
// |`%APPDATA%\@{product-short}\storage\\ada@example.com\Mail\NewLocalFolder`

// |Mac
// |`~/Library/Application Support/@{product-short}/ada@example.com/Mail/OldLocalFolder`
// |`~/Library/Application Support/@{product-short}/storage/ada@example.com/Mail/NewLocalFolder`
// |===

// ==== メールをインポートする

// . {product-short} のデスクトップアプリをアクセスします。
// . *Local Folder* の直下にあるフォルダをクリックします -- 実例の場合は`NewLocalFolder`です
// . フォルダ (`NewLocalFolder`) の上にマウスを置きますと、リフレッシュのアイコンが表示します。
// . リフレッシュのアイコンをクリックし、古いアプリからコピーしたメールが新しいフォルダ(`NewLocalFolder`)で閲覧することが可能となります。

// == ローカルのゴミ箱フォルダについて
// ローカルのゴミ箱フォルダは *Local Folders* で削除したメールやフォルダを保管します。ゴミ箱では、メールを閲覧済みにマークするか、フォルダ内のメッセージをすべて削除することが可能です。

// === 既読としてマークする
// このオプションでは、ローカルのゴミ箱フォルダにあるすべてのメールを既読としてマークします。


// . *Local Folders*の直下にある *Trash* を右クリックします。
// . メニューから *Mark emails as read* をクリックします。
// . すべてのメッセージが既読としてマークされます。

// === ゴミ箱フォルダを空にする
// このオプションでは、ローカルのゴミ箱フォルダにあるすべてのメールを削除します。

// . *Local Folders*の直下にある *Trash* を右クリックします。
// . メニューから *Empty trash* をクリックします。
// . すべてのメッセージが削除されます。

=== ローカルフォルダを管理する

// NOTE: この機能は Windows 用の {product-short} デスクトップアプリでのみ提供しています。

ローカルフォルダを活用することで、古いメールやサイズが大きいメールを削除せずに、クラウド上に割り当てた容量の上限を保護することが可能です。メールをクラウドからローカルのストレージへ移動し、メールをご利用中のコンピュータ上で保管することになります。

=== 複数ローカルフォルダを作成する
. 左側にて、*Local Storage* の上にマウスを置きます。
. image:graphics/plus.svg[plus icon, width=20] のアイコンをクリックし、新しいフォルダを作成します。
. フォルダ名を入力し、キーボードの Enter キーを押します。
. 新しく作成したフォルダをローカルストレージ上で確認できます。

=== ローカルフォルダのフォルダ名を変更する
. *Local Storage*の直下にあるフォルダを右クリックします。
. メニューから *Rename* をクリックします。
. フォルダの新しい名を入力します。
. キーボードの Enter キーを押すとフォルダ名が新しい名に変更されます。

=== ローカルフォルダをローカルのゴミ箱へ削除する
. *Local Storage*の直下にあるフォルダを右クリックします。
. メニューから *Delete* をクリックします。
. 確認画面にて、*OK* をクリックします。
. 削除したフォルダがローカルフォルダで *Trash* へ移動されます。
endif::Desktop_app[]
